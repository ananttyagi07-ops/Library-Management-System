# Library-Management-System
A simple Java-based Library Management System demonstrating core Object-Oriented Programming concepts such as classes, objects, constructors, methods, and association relationships. The project allows basic operations like displaying book/student details, issuing books, and returning books using clean and beginner-friendly Java code.

## Features 
- Add and display book details
- Add and display student details
- Issue books
- Return books
- Check whether a book has been already issued

 ## Technologies Used
 - Java
 - Object-Oriented Programming (OOP)

 ## Concepts Learned
   - Classes and objects
   - Methods
   - Encapsulation(basic-level)
   - Constructors
